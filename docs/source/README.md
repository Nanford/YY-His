# docs/source/ — 医学业务规则源文件（只读）

本目录四份文件是本 Demo 全部医学业务规则的**单一事实来源**，由医学侧提供，**禁止修改、重命名或覆盖**：

- `量表题目_Demo.txt` — 4 类评估的标准题目、选项、评分与判定规则
- `评估标签-干预标签知识图谱映射表_Demo.xlsx` — 17 评估标签 → 15 干预标签，75 条映射边
- `干预标签_Demo.xlsx` — 15 个干预标签的完整执行方案文本
- `老年健康智能评估与干预系统 Demo 需求文档.docx` — 总体需求

程序不直接读取这些文件，运行时数据见 `data/`（由 `npm run convert-rules` 生成/校验）。
